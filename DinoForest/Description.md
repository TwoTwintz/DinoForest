# DinoForest

## Description

Dans un monde en 2D un joueur doit survivre et elever des dinosaures dans sa base un monde infini générer aleatoirement avec au debut un portail pour aller dans le monde.
 -> Zone 1

Le joueur devra aussi explorer le monde infini générer aleatoirement avec plusieurs biomes ainsi au debut un portail pour aller a la base.
 -> Zone 2
